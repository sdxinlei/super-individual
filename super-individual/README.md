## 📦 项目简介

**超级个体** 是一个探索 AI 杠杆极限的实验项目。核心理念是：

> **一个人，不需要成为全栈工程师。只需要学会驾驭 AI 双引擎——Claude Code（架构师）和 Codex（执行者）——就能释放 3-5 人小团队的产出力。**

所有作品均由 Claude Code 设计架构 + Codex 执行编码。

### 双引擎协作模型

```
Claude Code (架构师)          Codex (执行者)
    ┃                             ┃
    ┣━ 需求分析 / 架构设计         ┣━ 单体函数/组件实现
    ┣━ 方案撰写 / 复杂逻辑推演     ┣━ 模板化代码生成
    ┣━ 代码审查 / 安全审计         ┣━ 脚本编写 / 测试编写
    ┣━ 任务拆解 / 跨文件评估       ┣━ 类型定义 / 文档生成
    ┗━ 最终集成验证               ┗━ 简单重构
              │
              ┗━━ 异步通信管道: brief.md → result.md
```
