## 三、内容创作模板

### 即刻帖子模板
```
标题：一句话钩子
正文：2-3 段，第一段场景/问题，第二段我的做法，第三段结果/数据
链接：工具链接
标签：#超级个体 #AI工具 #独立开发

要点：真实感 > 完美感，多说"我做了什么"少说"你应该"
```

### 小红书模板
```
封面：工具截图/使用场景（加文字标题）
正文：
1. 痛点引入（1-2句）
2. 我的解决方案（工具名+做什么）
3. 怎么用（简单3步）
4. 效果展示（截图）
5. 领取方式（链接/回复）

标签：#AI工具 #效率工具 #免费工具
```

### X/Twitter 模板
```
Hook line (问句/数据/反差)
1-2 句解释
Call to action
1-2 relevant hashtags

风格：简洁、英文、技术感
```

## 四、发布节奏 & 互动策略

| 时间 | 动作 |
|------|------|
| 早 8:00 | 发即刻/小红书 |
| 中 12:00 | 回复评论 + X/Twitter 发布 |
| 晚 20:00 | 互动（评论热门帖子、回复粉丝） |

### 互动原则
1. **不刷屏** — 每条内容间隔至少 4 小时
2. **每条回复必回** — 前 30 天每条评论都回复
3. **有价值评论** — 在别人帖子下写"补充信息"而非"写得真好"
4. **数据驱动** — 3 天后看数据，什么类型互动高就多做
