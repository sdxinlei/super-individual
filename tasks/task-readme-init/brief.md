# Task: task-readme-init - GitHub 仓库 README 与作品集初始化

## 背景
根据「超级个体计划-执行篇」Day 1&4 要求：GitHub 创建仓库 super-individual，README 写自我介绍 + 作品列表。为后续所有开源项目建立统一的品牌 README 模版。

## 输入
- 项目名: super-individual
- 计划角色: 超级个体实验室 / 独立开发者 / AI 双引擎协作（Claude Code × Codex）
- 已有作品: ① 个人主页（task-personal-site） ② 周报生成工具（task-weekly-report）

## 要求
1. 撰写 GitHub 仓库根目录 README.md 内容
2. 包含：项目简介、品牌定位、作品列表（含链接占位）、技术栈标签、联系方式
3. 风格：专业、技术向、有记忆点
4. 同时写一个统一的 GitHub Profile README（即同名仓库的 profile）模版
5. 需要 badges（可引用 shields.io）

## 产出文件
- tasks/task-readme-init/artifacts/README.md: 仓库 README
- tasks/task-readme-init/artifacts/PROFILE.md: GitHub Profile 模版（用户同名仓库使用）
- tasks/task-readme-init/artifacts/works.md: 作品集 Markdown（持续更新用）
- tasks/task-readme-init/result.md: 完成报告

## 注意事项
- 不要创建实际 GitHub 仓库，只产出文件
- 格式为纯 Markdown，用户可直接复制使用
- 作品链接使用占位符 {{url}} 方便用户替换
