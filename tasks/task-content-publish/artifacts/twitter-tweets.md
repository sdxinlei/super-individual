## 5. Dev Tip — Why I Use Claude Code as My Architect

Most devs use AI as a code generator.

I use it as an architect.

My workflow:
1. Describe the feature in plain English
2. AI breaks it into sub-tasks + writes the plan
3. I review the architecture before writing any code
4. AI handles the boring parts (tests, docs, edge cases)

Result: cleaner code, fewer rewrites, faster shipping.
