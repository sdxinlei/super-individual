# Task: task-xhs-saas - 小红书工厂 SaaS MVP 打磨

## 背景
小红书内容工厂 MVP 已完成，现需升级为可收费的 SaaS 工具。目标：用户可以免费试用 →  按量付费。

## 前提
- 已存在 `super-individual/xhs-factory/index.html` 和 `app.js`
- 深色科技风，靛蓝紫色系
- Claude API + Mock 降级

## 要求

### 1. 新增付费墙框架（纯前端模拟，不接真实支付）
- 免费用户：每天 3 次生成
- 高级用户（localStorage token 标记）：无限次
- 在 UI 中显示已用次数 / 剩余次数
- 超出限制时弹出「升级」页：列出三个档位（月/季/年），点击触发升级流程（localStorage 写入高级标记 + 跳转感谢页）

### 2. UI 升级
- 添加「定价」页面（/pricing 或内嵌弹窗）：
  - 基础版 ¥29/月：每天 10 次，5 种风格
  - 专业版 ¥79/月：每天 50 次，5 种风格 + 批量生成 10 条
  - 尊享版 ¥199/月：无限次，5 种风格 + 批量 + AI 优化建议
- 添加使用统计仪表盘（简单版）：今日生成次数、本月总计、剩余次数
- 首次访问引导弹窗：功能介绍 + 免费试用提示

### 3. 功能增强
- 历史记录：每次生成的结果存 localStorage（最多 50 条），可查看和复用
- 导出为图片：用 html2canvas 或 canvas 绘制分享图（可以先做占位按钮，实现存 base64）
- 增加「随机灵感」按钮：一键填入随机的热门话题示例

### 4. SEO 与分享
- 添加 Open Graph 标签
- 分享卡片预览（生成结果可分享为卡片）

## 产出
- `super-individual/xhs-factory/index.html` — 修改
- `super-individual/xhs-factory/app.js` — 修改（或新增 saas.js）

## 注意事项
- 不改原有 Mock 和 API 逻辑，只加功能
- 付费是模拟 UI，不接真实支付通道
- 与主站风格一致
