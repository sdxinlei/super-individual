# Task: task-xhs-factory - AI 小红书内容工厂 MVP

## 背景
现有 AI 文案助手能生成多平台文案，但小红书是独立开发者的黄金变现渠道。做一个专注小红书的"内容工厂"：标题生成 + 正文撰写 + 封面文案 + 标签推荐，一站式出稿。与文案助手互补，可作为独立收费工具。

## 输入
- 风格统一：深色科技风，靛蓝紫色系（与主站一致）
- API 复用：参考 `super-individual/copywriter/app.js` 的 Claude API 调用模式
- 部署方式：纯前端，GitHub Pages

## 架构设计

### 目录结构
```
super-individual/xhs-factory/
├── index.html    # 主页面
├── app.js        # 逻辑（API + Mock + UI）
└── style.css     # 样式（可选，也可以 inline）
```

### UI 流程
```
┌─ 输入区 ─────────────────────────────────┐
│  主题/关键词 (必填)                       │
│  产品/账号定位 (选填)                     │
│  目标受众 (选填)                          │
│  风格: [种草] [干货] [情感] [教程] [测评]  │
│  字数: [短(200字)] [中(500字)] [长(1000字)]│
│  数量: 1-5 条                             │
│  ┌──────────────────────────────────────┐ │
│  │ 🔑 API Key（可选，留空用本地模板）    │ │
│  └──────────────────────────────────────┘ │
│  [✨ 生成内容]                            │
├─ 输出区 ─────────────────────────────────┤
│  ┌─ 笔记 1 ────────────────────────────┐ │
│  │ 标题: xxxxx                          │ │
│  │ 正文: xxxxx...                       │ │
│  │ 封面文案: xxxxx                      │ │
│  │ 标签: #xxx #xxx #xxx                │ │
│  │ [复制全部] [复制标题] [复制正文]      │ │
│  └──────────────────────────────────────┘ │
└──────────────────────────────────────────┘
```

### 数据结构
```javascript
// 输入
{
  topic: "AI工具推荐",
  positioning: "效率类AI工具分享",
  audience: "职场白领",
  style: "种草",  // 种草|干货|情感|教程|测评
  length: "medium",  // short|medium|long
  count: 3
}

// 输出 (每条笔记)
{
  title: "打工人必备！这5个AI工具让我效率翻倍",
  body: "正文内容...",
  coverText: "效率翻倍的AI工具",
  tags: ["#AI工具", "#效率提升", "#打工人"]
}
```

### Prompt 模板设计
- **种草风格**：口语化、情绪化、"绝了""必入""后悔没早用"
- **干货风格**：清单式、数据化、教程感
- **情感风格**：故事化、共鸣感、第一人称
- **教程风格**：步骤清晰、图文对应
- **测评风格**：对比、优缺点、评分

### API 调用
复用文案助手模式：
- Claude API `/v1/messages`
- API Key localStorage 持久化
- 失败降级 Mock

## 产出文件
- `super-individual/xhs-factory/index.html` — 主页面（全部 inline style + 引用 app.js）
- `super-individual/xhs-factory/app.js` — 独立逻辑文件

## 质量要求
- 5 种风格 Prompt 各有不同
- Mock 模板每种风格至少 3 条
- 每条笔记含标题/正文/封面文案/标签 4 个部分
- 复制功能：复制全部、复制标题、复制正文
- 与主站风格一致（深色科技风，靛蓝紫色系）
- 导航栏链接回主站
