# Task: content-xhs - 小红书笔记批量生产（15篇）

## 背景
超级个体品牌内容生产。小红书风格：有用且好看，标题钩子强，正文有结构。

## 要求
生成 15 篇小红书笔记，分 5 个主题，每个主题 3 篇。

### 主题矩阵

**1. AI效率工具推荐（3篇）**
- 独立开发者篇（周报生成器+Claude Code）
- 自媒体篇（文案助手+小红书工厂）
- 职场人篇（AI 提效工作流）

**2. 副业/赚钱（3篇）**
- AI 赋能自由职业（具体案例）
- 一个人怎么接单
- 副业心态建设

**3. 小红书运营技巧（3篇）**
- 标题公式
- 标签策略
- 排版与视觉

**4. 工具种草（3篇）**
- 小红书工厂深度测评
- AI 文案助手使用教程
- 周报生成器黑科技

**5. 心态/成长（3篇）**
- 裸辞做独立开发
- 30 岁重启人生
- 自律与自由

### 每篇格式
```
标题：hook标题（带数字/痛点/好奇）

正文：800-1200字
- 开头：痛点/故事（100字）
- 中间：3-5个要点，分段+emoji
- 结尾：行动号召或评论区互动

标签：#tag1 #tag2 #tag3 #tag4 #tag5

封面卡标题：10字以内，醒目
封面卡副标题：15字以内
```

### 视觉
每篇需要指定封面卡标题和副标题（用于 cover-template.html），参考格式：
```
封面卡标题：效率翻3倍的AI工具
封面卡副标题：独立开发者私藏清单
```

### 红线
- ❌ 不能和之前 3 篇重复
- ❌ 没有具体数据不行
- ❌ 语气要真实
