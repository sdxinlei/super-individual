# Task: content-jike - 即刻帖子批量生产（20条）

## 背景
超级个体品牌的社交媒体内容生产。即刻平台风格：技术接地气、真实故事感、口语化。

## 要求
生成 20 条即刻帖子，分 5 个主题，每个主题 4 条。

### 主题矩阵

**1. 独立开发日常（4条）**
- bug 故事、熬夜debug、上线紧张瞬间、用户第一条好评
- 语气：真实自嘲，有故事感

**2. 工具推荐（4条）**
- AI文案助手、小红书工厂、周报生成器 3 个工具轮流推
- 每条突出一个具体功能点

**3. AI 工作流（4条）**
- Claude Code 当架构师、Codex 当执行者、双引擎怎么配合
- Prompt 技巧、效率对比

**4. 创业思考（4条）**
- 超级个体理念、一人公司、不融资、AI 杠杆
- 数据说话：30天3个工具、0服务器成本

**5. 技术分享（4条）**
- 纯前端模式、API Key 模式、0 运维
- 优缺点都讲，显得真实

### 每条格式
```
【hook 标题】

正文 80-150 字
口语化，有具体故事/数据
不要"很高效"要"提了3倍"

#超级个体 #独立开发 #AI
```

### 红线
- ❌ 不能和 task-content-publish 已有的 5 条重复
- ❌ 不能泛泛而谈
- ❌ 不能有错别字
