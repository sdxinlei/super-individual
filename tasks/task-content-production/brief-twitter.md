# Task: content-twitter - X/Twitter 推文批量生产（15条）

## 背景
超级个体品牌全球发声。X 平台风格：简洁、专业、有洞察。

## 要求
生成 15 条英文推文，分 3 个主题，每个主题 5 条。

### Building in Public（5条）
- 个人实验数据、进展
- 超级个体理念
- 要具体："3 tools in 30 days"而不是"many tools"

### Dev Tips（5条）
- 纯前端架构技巧
- Claude Code 使用心得
- API Key 模式优缺点

### Tool Launches（5条）
- 文案助手/小红书工厂/周报生成器 轮流发
- 每条突出一个独特卖点

### 每条格式
<200 chars，英文
含 2-3 个相关标签
配图建议（如截图/封面卡）

### 红线
- ❌ 不能和已有 5 条重复
- ❌ 不能低于 120 chars（太短没内容）
- ❌ 不能有语法错误

## 产出
`tasks/task-content-production/artifacts/twitter-15.md`

## 自检
每条检测：
1. <200 chars？□
2. 有具体信息不空洞？□
3. 有标签？□
