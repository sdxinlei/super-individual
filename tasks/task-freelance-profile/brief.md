# Task: task-freelance-profile - 接单平台 Profile 与服务描述生成

## 背景
超级个体需要在 Fiverr、电鸭社区、猪八戒三个接单平台发布服务。现有 profile 已注册但服务描述和 profile 内容需要完善。Codex 负责生成各平台的完整 profile 文案和服务描述 Markdown 文件，用户手动复制到各平台。

## 输入
- 个人品牌定位：AI 工具开发者 | 独立开发者 | 全栈交付
- 现有主页：https://super-individual.com/
- 已有作品：AI 周报生成器、AI 文案助手
- 技术栈：Python, JavaScript, HTML/CSS, Claude AI, OpenAI API

## 要求

### 产出物 1: Fiverr Profile
文件：`tasks/task-freelance-profile/artifacts/fiverr-profile.md`
- Gig 标题（英文）：3-5 个选项
- Gig 描述（英文）：300-500 字，含关键词 SEO
- 服务定价：Basic ($50) / Standard ($150) / Premium ($350)
- 服务类目：AI Automation, Web Development, Chatbot Development
- FAQ：5 个常见问题
- 要求：参考 Fiverr 同类 Top Seller 的写法风格

### 产出物 2: 电鸭 Profile
文件：`tasks/task-freelance-profile/artifacts/dianya-profile.md`
- 个人简介（中文）：200 字以内
- 服务描述：AI 自动化工具开发、API 集成、爬虫脚本
- 定价策略：基础 500 元起 / 标准 1500 元 / 高级 3000 元
- 案例展示：周报生成器 + 文案助手 + 个人主页
- 接单流程说明

### 产出物 3: 猪八戒 Profile
文件：`tasks/task-freelance-profile/artifacts/zbj-profile.md`
- 店铺简介（中文）
- 服务分类：AI 工具开发 / 自动化脚本 / Web 应用
- 服务定价

## 注意事项

- 突出"AI 双引擎（Claude Code + Codex）"的差异化优势
