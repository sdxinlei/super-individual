# 超级个体计划：从 0 到 100 万（12个月）

**用户画像**：综合技能起步期 | 半全职（6-8h/天）| 资金 0-1 万 | 不限方向

---

## 🎯 核心策略：AI 杠杆 × 快速验证 × 三条腿走路

你的 **超级武器** 是你同时拥有 Claude Code（战略+架构）和 Codex（执行）两个 AI 搭档。一个人 = 一个 3-5 人小团队的生产力。

```
     ┌──────────────┐
     │  超级个体     │
     ├──────────────┤
     │ Claude Code  │── 战略/架构/审查/方案
     │    + Codex   │── 执行/编码/测试/并行
     ├──────────────┤
     │ 效率 = 3-5人 │
     └──────────────┘
```

### 三条收入腿（缺一不可）

```
收入流 1: AI 项目外包/咨询     ── 现金流（短期）
收入流 2: AI 工具/SaaS/出海     ── 复利收入（中期）
收入流 3: 知识资产（课程/IP）    ── 杠杆收入（长期）
```

---

## 📅 四阶段执行计划

### 🔵 阶段 0：准备期（第 1-2 周）

**目标**：找到切入点、搭建基础设施、跑通第一个 AI 工作流

| 周 | 行动 | AI 协作方式 |
|----|------|------------|
| W1 | **定位诊断**：列出 5 个你能用 AI 做的方向，选 1 个主攻 | 我做分析 & 拆市场 |
| W1 | **作品集搭建**：用 AI 生成 2-3 个 demo 项目放 GitHub | Codex 执行生成 |
| W2 | **个人品牌冷启动**：在即刻/小红书/X 开始日更 AI 相关内容 | 我定内容策略 |
| W2 | **接单渠道搭建**：注册 upwork/fiverr/猪八戒/电鸭/朋友圈发声 | 都做 |

**交付物**：
- ✅ 个人作品集站点（`super-individual.com` 或 GitHub Pages）
- ✅ 至少 5 条 AI 相关内容发布
- ✅ 跑通一个 AI 项目交付的全流程

---

### 🟢 阶段 1：生存期（第 1-3 个月）

**目标**：月入 1-2 万，找到 PMF （Product-Market Fit）

**收入构成**：
```
AI项目外包    ─── 80%（1-2万/月）
Side Product  ─── 15%（试水阶段）
内容变现     ───  5%（早期）
```

| 月 | 关键行动 | 预期收入 |
|----|---------|---------|
| M1 | **扫单模式**：接 AI 自动化/爬虫/工具开发外包，低价快速积累案例。单价 3-5k，月接 3-4 单 | 1-1.5万 |
| M1 | **同时启动 1 个微型工具**：用 1 周时间 + AI 做一个解决具体问题的小工具（eg. AI 周报生成器、小红书文案工具） | 免费引流 |
| M2 | **提高客单价**：第一个月案例在手，单价提到 5-8k，月接 2-3 单 | 1.5-2万 |
| M2 | **工具产品上线收费**：微工具加付费墙（月费 29-49 元），目标 50-100 用户 | 0.2-0.5万 |
| M3 | **筛选高价值客户**：放弃低单价客户，聚焦 AI 咨询+定制开发，单均 1-2万 | 2-3万 |
| M3 | **复盘方向**：哪个工具/服务方向有感觉 → 加注 | - |

**Codex 在这里的作用**：
```
项目外包中 80% 的编码工作 → Codex 执行
├─ 爬虫/数据清洗脚本     → Codex 直接写
├─ AI API 集成           → Codex 写调用代码
├─ 前端页面/后台管理界面  → Codex 生成
├─ 测试                  → Codex 写
└─ 我：架构设计 + 方案 + 客户沟通 + 交付把关
```

**关键指标**：✅ 月入稳定过万 ✅ 找到 1-2 个高意向方向 ✅ 微型工具有付费用户

---

### 🟡 阶段 2：放大期（第 4-6 个月）

**目标**：月入 3-5 万，确立核心赛道

**收入构成**：
```
AI项目/咨询    ─── 50%（1.5-2.5万/月）
工具产品      ─── 30%（1-1.5万/月）
出海尝试      ─── 15%
知识资产      ───  5%
```

| 月 | 关键行动 | 预期收入 |
|----|---------|---------|
| M4 | **聚焦 1 个赛道**：从 M1-M3 中选 MVP 方向，all-in | 2-3万 |
| M4 | **工具产品 2.0**：基于用户反馈大迭代，定价提到 49-99/月 | - |
| M5 | **出海尝试**：如果工具是通用需求 → 上 Product Hunt，出海收美金 | 3-4万 |
| M5 | **启动知识资产**：过程沉淀一套《从 0 用 AI 做产品》内容体系 | 蓄力 |
| M6 | **构建被动收入管道**：工具产品+出海+内容开始自动获客 | 4-5万 |

**关键转变**：从"卖时间"转向"卖产品"。时间单价从 5k → 1-2 万/单，同时产品开始自动贡献收入。

**Codex 在这里的升级作用**：
```
├─ 并行维护 2-3 个工具产品    → Codex 处理日常迭代
├─ 出海所需的 landing page    → Codex 生成多语言版本
├─ 项目外包继续跑            → Codex 主力编码
├─ 自动化测试/CI/CD          → Codex 搭建
└─ 我：产品方向 + 架构 + 增长策略 + 关键决策
```

**关键指标**：✅ 月入稳定 4 万+ ✅ 工具产品月 recurring 1万+ ✅ 找到出海切入点

---

### 🟠 阶段 3：规模化期（第 7-9 个月）

**目标**：月入 6-8 万，建立系统

**收入构成**：
```
工具产品/出海  ─── 50%（3-4万/月）
高价值咨询    ─── 30%（2-2.5万/月）
知识付费      ─── 15%（1万+/月）
投资收入      ───  5%
```

| 月 | 关键行动 | 预期收入 |
|----|---------|---------|
| M7 | **产品矩阵化**：主产品 + 2 个衍生品（不同定价），MRR 目标 3 万 | 5-6万 |
| M7 | **知识付费上线**：小报童/知识星球/录播课，定价 299-999，目标 50-100 人 | - |
| M8 | **减产提价**：月接不超过 2 个项目，单均 2-3 万 | 6-7万 |
| M8 | **出海产品进入增长期**：SEO + 内容营销 + 付费广告小规模测试 | - |
| M9 | **系统化**：SOP 化，减少自己动手时间，腾出精力思考更大布局 | 7-8万 |

**Codex 作用**：
```
├─ 产品日常维护 & 客服自动回复 → Codex 全权处理
├─ 知识付费内容整理/排版      → Codex 生成
├─ SEO 内容批量产出           → Codex 写内容
├─ 项目外包部分独立交付        → Codex 独立完成简单项目
└─ 我：产品战略 + 增长 + 品�� + 高价值单
```

**关键指标**：✅ 月入稳定 7 万+ ✅ MRR 3 万+ ✅ 知识付费跑通 ✅ 减少自身劳动时间到 50%

## 🚨 风险对冲

| 风险 | 概率 | 应对 |

| 外包接不到单 | 中 | 降价 + 铺渠道 + 做开源项目吸引注意力 |

| AI 能力不足以交付 | 低 | 我做复杂部分，Codex 做简单部分，分工明确 |
